# Python Password Generator

This is a simple Python project that generates a random password based on user input.

## Features
- User enters password length
- Generates random password
- Uses letters, numbers, and symbols

## Technologies Used
Python

## How to Run

1. Clone the repository
2. Open the folder
3. Run the program

python main.py

## Example

Enter password length: 8  
Generated Password: aB9@kP2!
